# Calculate distance between trees, or lists of trees

Calculate distance between trees, or lists of trees

## Usage

``` r
.TreeDistance(Func, tree1, tree2, checks = TRUE, ...)
```

## Arguments

- checks:

  Logical specifying whether to perform basic sanity checks to avoid
  crashes in C++.

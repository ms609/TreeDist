# List clades as text

List clades as text

## Usage

``` r
ReportMatching(splits1, splits2, realMatch = TRUE)
```

## Arguments

- splits1, splits2:

  Logical matrices with columns specifying membership of each
  corresponding matched clade.

## Value

`ReportMatching` returns a character vector describing each pairing in a
matching.
